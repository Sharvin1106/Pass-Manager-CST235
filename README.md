## Security and Reliability Features Implemented
- Encryption and Decryption of Information
- Backup and Recovery
- Security PIN
- Email Verification
- Biometric Authentication
- Multistep Login
- Two factor authentication
- Screenshot feature locked
- Certification Pinning
- Session Timeout
- Captcha
- Event Logging

    </ul>
 </ol>
